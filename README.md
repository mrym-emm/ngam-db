## üìä Data Flow Summary

### **User Journey Flow**
1. **Browse Threads** ‚Üí User sees communities/categories  
2. **View Listings** ‚Üí User clicks thread, sees items for sale/wanted  
3. **AI Matching** ‚Üí System generates matches based on compatibility  
4. **View Matches** ‚Üí User sees recommended listings  
5. **Contact Seller** ‚Üí User sends message  
6. **Ask Questions** ‚Üí User posts FAQ on listing  
7. **AI Chat** ‚Üí User asks AI for help finding items  
8. **Track Activity** ‚Üí System logs user actions  
9. **Earn Achievements** ‚Üí System unlocks badges  

---

### **Key Database Interactions**
| Operation | Purpose |
|------------|----------|
| ‚úÖ **READ** | Browse, search, view matches |
| ‚úÖ **WRITE** | Create listings, send messages, post FAQs |
| ‚úÖ **UPDATE** | Mark messages as read, update stats |
| ‚úÖ **JOIN** | Get listings with user info, matches with details |
| ‚úÖ **AGGREGATE** | Count views, calculate ratings, stats |

## üîó Cross-Table Queries

### üßç Get Listings with Owner Info
```sql
SELECT 
  l.*,
  u.full_name AS seller_name,
  u.seller_rating,
  u.is_verified AS seller_verified,
  u.avatar_url AS seller_avatar
FROM listings l
JOIN users u ON l.user_id = u.id;

### Get Matches with Listing Details

```sql
SELECT 
  m.*,
  ml.title, ml.price, ml.location, ml.image_url,
  u.full_name, u.seller_rating, u.is_verified
FROM listing_matches m
JOIN listings ml ON m.matched_listing_id = ml.id
JOIN users u ON ml.user_id = u.id;
```

### Get FAQs with Nested Replies

```sql
WITH RECURSIVE faq_tree AS (
  SELECT f.*, a.* FROM listing_faqs f
  LEFT JOIN faq_answers a ON f.id = a.faq_id
  -- ... recursive logic
);
```

### Get Conversation Messages

```sql
SELECT 
  m.*,
  u.full_name, u.avatar_url, u.is_verified
FROM messages m
JOIN users u ON (m.sender_id = u.id OR m.recipient_id = u.id)
WHERE m.sender_id = ? OR m.recipient_id = ?;
```

### Get AI Chat History

```sql
SELECT 
  s.id, s.title,
  array_agg(m.*) AS messages
FROM ai_chat_sessions s
LEFT JOIN ai_chat_messages m ON s.id = m.session_id
GROUP BY s.id;
```
